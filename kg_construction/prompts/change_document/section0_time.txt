## 属性描述

**time（时间）**：
- 会议日期或变更日期
- 格式：YYYY年MM月DD日 或 YYYY-MM-DD

**提取要点**：
- 查找"2023年08月02日"这种格式的日期
- 优先提取会议日期
- 如果有多个日期，提取第一个

## 输出格式

请以JSON格式返回：

```json
{
  "nodes": [
    {
      "node_type": "时间",
      "attributes": {
        "time": "提取的会议时间，格式如2023年08月02日"
      }
    }
  ]
}
```

**注意事项**：
- 如果文本中没有时间信息，返回 `{"nodes": []}`
- 不要提取里程范围信息
